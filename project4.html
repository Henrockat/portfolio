LED Binary Op En Neer Teller

Deze teller toont een binaire waarde via LED's en laat toe om deze op en neer te tellen met twee knoppen. Een potmeter regelt de helderheid van de LED's.

Mogelijkheid tot simulatie

Je kunt deze schakeling ook virtueel testen op Wokwi.com, een online Arduino en embedded simulator.

Benodigde hardware

  • 1x Raspberry Pi Pico W
  • 6x LED (2 rood, 2 geel, 2 groen)
  • 6x Weerstanden (220Î©)
  • 2x Drukknoppen
  • 1x Potentiometer
  • Breadboard
  • Jumper wires
  • Micro USB-kabel

Stappenplan / Tutorial

  1. Verbind de LED's via weerstanden met 6 GPIO-pinnen van de Pico W.
  2. Verbind de potmeter met een analoge pin voor helderheidsregeling.
  3. Sluit twee knoppen aan voor het optellen en aftellen.
  4. Upload de code via Thonny of een ander MicroPython-platform.
  5. Bekijk het binaire resultaat op de LED's, afhankelijk van de knop die je indrukt.

Schematische weergave

MicroPython Code (Raspberry Pi Pico)


from machine import Pin, ADC, PWM
from time import sleep

led_pins = [Pin(i, Pin.OUT) for i in [2, 3, 4, 5, 6, 7]]
button_up = Pin(8, Pin.IN, Pin.PULL_DOWN)
button_down = Pin(9, Pin.IN, Pin.PULL_DOWN)
pot = ADC(26)

value = 0

def display_binary(val, brightness):
    for i in range(6):
        led = PWM(led_pins[i])
        led.freq(1000)
        duty = brightness if (val >> i) & 1 else 0
        led.duty_u16(duty)

while True:
    brightness = pot.read_u16()
    display_binary(value, brightness)
    if button_up.value():
        value = (value + 1) % 64
        sleep(0.2)
    elif button_down.value():
        value = (value - 1) % 64
        sleep(0.2)
