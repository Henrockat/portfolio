LED Binary Op En Neer Teller met Raspberry Pi Pico

Deze teller toont een binaire waarde via LED's en laat toe om op en neer te tellen met twee knoppen. Een potentiometer regelt de helderheid van de LED's.

Mogelijkheid tot simulatie

Je kunt deze schakeling virtueel testen op Wokwi.com. Wokwi ondersteunt nu ook de Raspberry Pi Pico met MicroPython, wat ideaal is om je code veilig te testen zonder hardware. Je kunt componenten zoals LED‚Äôs, knoppen en potentiometers eenvoudig slepen en verbinden in een interactieve simulatieomgeving.

Benodigde hardware

  • 1x Raspberry Pi Pico W
  • 6x LED (kleur naar keuze)
  • 6x Weerstanden (220Œ©)
  • 2x Drukknoppen
  • 1x Potentiometer
  • Breadboard
  • Jumper wires
  • Micro USB-kabel

Stappenplan / Tutorial

  1. Verbind de LED's via weerstanden met GPIO-pinnen op de Pico.
  2. Sluit de potentiometer aan op een analoge ingang (vb. GP26).
  3. Verbind de drukknoppen met twee digitale GPIO-pinnen (GP8 en GP9).
  4. Upload de MicroPython-code met behulp van Thonny of via VS Code met de Pico extensie.
  5. Bekijk hoe de binaire teller reageert op input en helderheid aanpast via de potmeter.
