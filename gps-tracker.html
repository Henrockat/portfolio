GPS Tag met TTGO T-Beam en LoRa

Deze GPS Tag op basis van de TTGO T-Beam maakt het mogelijk om fietsen, rugzakken of andere waardevolle voorwerpen op afstand te volgen. Dankzij GPS en LoRa-communicatie kun je live locatiegegevens versturen zonder mobiel netwerk.

Wat kun je ermee?

Dit project toont hoe je een low-power GPS-tracker maakt die via LoRa en The Things Network (TTN) gegevens kan verzenden. Enkele mogelijke toepassingen:

  • Fiets- of voertuigtracking
  • Locatiebewaking van bagage of huisdieren
  • Experimentele IoT-projecten met LoRaWAN
  • Werken met ESP32, GPS, en TTN in √©√©n compact systeem

Schematische weergave

Doel van het project

In dit project leer je hoe je GPS-gegevens verzamelt en via LoRa verzendt naar een netwerk zoals TTN. Je leert hoe je de ESP32 TTGO T-Beam configureert en hoe je C++ code compileert en aanpast in de Arduino IDE.

Stappenplan / Tutorial

  1. Sluit de TTGO T-Beam aan op je computer via USB.
  2. Installeer ESP32 board support in Arduino IDE.
  3. Installeer benodigde libraries: TinyGPS++, LMIC, en LoRa.
  4. Kloon of download de repository van GitHub:
  5. ttgo-tbeam-ttn-tracker
  6. Open het bestand ttn-tracker.ino en vul in config.h je TTN keys in (AppEUI, DevEUI, AppKey).
  7. Compileer en upload de code naar je T-Beam.
  8. Bekijk via de Serial Monitor of de tracker verbinding maakt en GPS-data verzendt.
