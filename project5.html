LED Lichtshow met Raspberry Pi Pico

Deze LED-show schakelt tussen vier effecten: links afrollen, rechts afrollen, fade-in chase en regenval-effect. De snelheid wordt geregeld via een potentiometer en via twee knoppen kies je eenvoudig het gewenste patroon.

Mogelijkheid tot simulatie

Je kunt deze schakeling virtueel testen op Wokwi.com. Wokwi ondersteunt ook Raspberry Pi Pico met MicroPython. Je kunt componenten zoals LED‚Äôs, knoppen en potmeters eenvoudig simuleren en je code direct uittesten zonder hardware.

Stappenplan / Tutorial

  1. Verbind de 8 LED‚Äôs met GPIO-pinnen via weerstanden.
  2. Sluit de potentiometer aan op een analoge poort (vb. GP26).
  3. Verbind twee knoppen met digitale ingangen (vb. GP10 en GP11).
  4. Upload de MicroPython-code via Thonny of Visual Studio Code met Pico-extensie.
  5. Gebruik de knoppen om tussen effecten te schakelen en de potmeter om de snelheid aan te passen.
