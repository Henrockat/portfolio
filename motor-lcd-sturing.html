DC-motorsturing met H-brug

In dit project leer je hoe je twee DC-motoren kan aansturen met een Arduino Nano via een L9110 H-brug module. De snelheid wordt bepaald via een potentiometer, en de richting via drukknoppen. In de Wokwi-simulatie worden LEDs gebruikt om de werking van de motoren visueel voor te stellen.

Mogelijkheid tot simulatie

In Wokwi werd deze opstelling gesimuleerd met LEDs i.p.v. DC-motoren. De LED-kleur en -intensiteit visualiseren richting en snelheid. Handig als je geen fysieke H-brug of motoren beschikbaar hebt.

Benodigde hardware

  • 1x Arduino Nano
  • 1x L9110 H-brug module
  • 2x DC-motoren
  • 2x Drukknop
  • 1x Potentiometer
  • 1x I2C LCD-scherm
  • Breadboard en jumper wires
  • USB-kabel

Stappenplan / Tutorial

  1. Sluit de motoren aan op de H-brugmodule en verbind deze met de PWM-uitgangen van de Arduino.
  2. Gebruik analogRead() om de stand van de potentiometer uit te lezen.
  3. Gebruik map() om deze waarde om te zetten naar een duty-cycle voor PWM.
  4. Stuur via analogWrite() de snelheid van de motoren aan.
  5. Gebruik drukknoppen om de richting te veranderen en geef de status weer op een I2C LCD.

Doel van het project

In dit project leer je hoe je op een gecontroleerde manier analoge input (zoals een potentiometer) gebruikt om een PWM-signaal aan te sturen. Daarnaast pas je interrupt-achtige logica toe om richtingswijzigingen via knoppen door te voeren. Via digitale leerboeken werk je stap voor stap en test je zowel in simulatie (Wokwi) als in een echte fysieke opstelling met motoren.

Wat ik geleerd heb

Dankzij dit project begrijp ik nu hoe een analoge input zoals een potentiometer uitgelezen wordt met analogRead(), en hoe deze waarde vertaald wordt naar een PWM-signaal via analogWrite() om de snelheid van een motor aan te passen.

Ik ontdekte hoe een L9110 H-brug werkt en hoe je via digitale pinnen de draairichting van een DC-motor bepaalt. Door te werken met zowel simulatie als echte hardware werd ik geconfronteerd met praktische aandachtspunten, zoals voedingsproblemen en correcte aansluiting.

Ook heb ik geoefend met het gebruik van een I2C LCD en de functie lcd.print() om informatie realtime zichtbaar te maken. De combinatie van theorie, simulatie en praktijk heeft mijn begrip van PWM, ADC en motorsturing enorm versterkt.
