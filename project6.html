GPS Tag TTGO T-Beam

Deze GPS Tag op basis van de TTGO T-Beam maakt het mogelijk om fietsen, rugzakken en andere waardevolle spullen in realtime te volgen. Dankzij GPS en LoRa-communicatie kun je eenvoudig de locatie traceren, zelfs over lange afstanden zonder gebruik van mobiel internet.

Schematische weergave

Benodigde hardware

  • 1x TTGO T-Beam module (met ingebouwde GPS en LoRa)
  • 1x USB-kabel (micro-USB)
  • 1x 18650 Li-Ion batterij (optioneel voor mobiele voeding)
  • GPS-antenne en LoRa-antenne
  • Computer met Arduino IDE geïnstalleerd

Stappenplan / Tutorial

  1. Sluit de TTGO T-Beam via USB aan op je computer.
  2. Installeer de juiste board definitions en libraries in de Arduino IDE (ESP32, TinyGPS++, LoRa libraries).
  3. Upload de GPS-tracking sketch naar het bord.
  4. Monitor de seriële output om GPS-positiegegevens te bekijken of zend ze via LoRa.

Voorbeeld Arduino Code


#include 
#include 

TinyGPSPlus gps;
HardwareSerial GPSSerial(1);

void setup() {
  Serial.begin(115200);
  GPSSerial.begin(9600, SERIAL_8N1, 34, 12); // RX=34, TX=12 (controleer je eigen T-Beam versie)
}

void loop() {
  while (GPSSerial.available() > 0) {
    gps.encode(GPSSerial.read());
    if (gps.location.isUpdated()) {
      Serial.print("Latitude: ");
      Serial.println(gps.location.lat(), 6);
      Serial.print("Longitude: ");
      Serial.println(gps.location.lng(), 6);
    }
  }
}
